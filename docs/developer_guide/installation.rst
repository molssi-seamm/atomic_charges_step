============
Installation
============


Stable release
--------------

To install the Atomic Charges Step, run this command in your terminal:

.. code-block:: console

    $ pip install atomic_charges_step

This is the preferred method to install Atomic Charges Step, as it will always
install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Atomic Charges Step can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/molssi-seamm/atomic_charges_step

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/molssi-seamm/atomic_charges_step/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .

from inside the top-level directory of the source code.    


.. _Github repo: https://github.com/molssi-seamm/atomic_charges_step
.. _tarball: https://github.com/molssi-seamm/atomic_charges_step/tarball/master
